Who Would Win?

"Who Would Win?" is a online web application produced using Javascript, Firebase and HTML/CSS.

The website allows users to select from various sports people, who would win. Then presenting results based on other users suggestions.

Each collection of results, showcases the random selections users have made relating to that individual person, as opposed to themselves in general therefore providing technical challenges from a development perspective.

Space Race
